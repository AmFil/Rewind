The Code for Rewind


                    //Get the string from the page
//controler function
function getValues() {

    document.getElementById("alert").classList.add("invisible");

    let userString = document.getElementById("userString").value;

    let revString = reverseString(userString); 

    displayString(revString); 

}

//Reverse the string
//logic function
function reverseString(userString) {

    let revString = []; 

    //reverse a string using a for loop
    for(let i = userString.length-1; i >= 0; i--) {
        revString += userString[i]; 
    }

    return revString;

}

//Display the reversed string to the user
//View Fuction
function displayString(revString) {

    //write to the page
    document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`; 
    //show the alert box
    document.getElementById("alert").classList.remove("invisible");
}

The Code is structured in three functions.

What each function does:

getValues

The function called "getValues" uses javascript to add a class called "invisible" that prevents an element with an id of "alert" from appearing. Next, a variable called "userString" is declared, this variable is defined when the user types in a string into the element with an id of "input". This input form can be seen on "The App" page. A variable called "revString" is declared and it uses the "reverseString" function which takes "userString" as an argument. Next, the "display" string function is called which uses the "revString" variable as an argument.

reverseString

The reverseString function declares an array called "revString" and uses "userString" in conjunction with a for loop to populate the array with every letter that comes from "userString" but in reverse order. This is done using the for loop. The for loop takes the last letter of the string and every letter before the last letter that comes from "userString" and makes it the first letter, second letter, etc.

displayString

displayString takes "reverseString as an argument. displayString takes the string that was reversed and injects it in between the tags of the element with the "msg" id. Although this has been done, the reverse string still cannot be seen. The next line in this function removes the "invisible" class that was inserted into the element with the "alert" id using the "getValues" function. Once, "invisible" has been removed, the reversed string appears along with a prompt that says, "Your string reversed is:". This is done using string interpolation.
